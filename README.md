# odin-recipes

This project *Project: Recipes* is the part of _Foundations Course_ of [The Odin Project](https://www.theodinproject.com/lessons/foundations-recipes)

This project is a simple project based on basic html. It requires to write 4 html pages, one of index and other three of any recipies.

## Content
This project contains four html pages:
1. index.html
2. rasgullas.html
3. kheer.html
4. chili-paneer.html

View the html pages [here](https://24f3002643.github.io/odin-recipes/)
